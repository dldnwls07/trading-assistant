# � Trading Assistant 시스템 고도화 및 다국어 지원 계획

사용자 피드백을 바탕으로 웹 앱의 분석 전문성을 높이고, 글로벌 서비스를 위한 4개국어 지원 기능을 추가합니다.

## 🛠️ 주요 작업 내용

### 1. 백엔드 분석 엔진 고도화 (`src/agents/analyst.py`)
*   **단/중/장기 관점 세분화**:
    *   **단기(1~5일)**: 60분봉 중심, RSI/MACD 골든크로스 가중치 부여.
    *   **중기(1~3개월)**: 일봉 중심, 20/50일 이평선 및 OBV 수급 분석.
    *   **장기(6개월+)**: 주봉/재무 중심, 200일선 및 PER/PBR 가치 평가.
*   **언어 지원**: 분석 리포트 생성 시 언어 파라미터를 받아 해당 언어로 텍스트 생성 지원.

### 2. 포트폴리오 분석 시스템 개선 (`src/agents/portfolio_analyzer.py`)
*   **상관계수 분석**: 보유 종목 간의 가격 움직임 상관도를 계산하여 포트폴리오의 실질적인 분산 효과 측정.
*   **리밸런싱 로직 강화**: 단순 수익률이 아닌 '리스크 대비 수익률' 관점의 제안 생성.

### 3. 종목 추천 스크리너 고도화 (`src/agents/screener.py`)
*   **추천 엔진 개선**: 코스피/나스닥 시가총액 상위 종목뿐만 아니라 섹터 주도주를 실시간 탐색하여 추천.
*   **스타일 매칭**: 공격적 성장, 배당, 가치 등 투자자 성향과의 일치도를 더 정교하게 산출.

### 4. 프론트엔드 다국어 지원 (Web App)
*   **i18n 도입**: `react-i18next` 라이브러리를 사용해 UI 텍스트(한, 영, 중, 일) 처리.
*   **언어 선택 드롭다운**: 헤더에 국기 아이콘이 포함된 언어 선택 메뉴 추가.
*   **UI 통일**: 현재 섞여 있는 언어를 설정된 언어로 완전 통일.

### 5. AI 채팅 및 분석 대응
*   **AI 프롬프트 수정**: Gemini API 호출 시 시스템 프롬프트에 사용자의 언어 설정을 포함하여 응답 언어 강제.
