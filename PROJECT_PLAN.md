# AI Trading Assistant Project Plan

## Architecture (V3.0 - Premium Web)
- **Backend:** Python (FastAPI) + yfinance + AI (FinBERT/LLM)
- **Frontend:** React (Vite) + TradingView Lightweight Charts + Tailwind CSS
- **Design:** Modern Fintech Dark Mode (inspired by Danelfin/Fint)

## Core Features
1. **Smart Analysis:** Daily(Trend) + Hourly(Entry) combined logic.
2. **Interactive Chart:** Professional candlestick charts with zooming.
3. **AI Alpha Signal:** Confidence-based buy/sell signals (weighted scoring).
4. **Smart Report:** AI-generated consulting reports translated into premium UI cards.

## Run Instructions
1. Run `start_web.bat` in the root folder.
2. Open `http://localhost:5173`.
3. Enter a ticker (e.g., AAPL) and see the magic.
