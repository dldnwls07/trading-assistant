# 📊 Trading Assistant 사용 가이드

## 🚀 실행 방법

### 방법 1: 더블클릭 (권장)
프로젝트 폴더의 **`실행.bat`** 파일을 더블클릭하세요.

### 방법 2: 터미널
```bash
python src/ui/overlay.py
```

---

## 🎯 사용법

1. **프로그램 실행** → 화면 좌측 상단에 분석 창이 뜹니다
2. **티커 복사** → 웹사이트나 트레이딩뷰에서 종목 코드를 복사 (`Ctrl+C`)
3. **자동 분석** → 프로그램이 자동으로 분석을 시작합니다
4. **결과 확인** → 매수/매도 신호, 타점, 상세 분석을 확인하세요

## 📈 분석 결과 해석

### 신호 종류
| 신호 | 점수 | 의미 |
|------|------|------|
| 🔥 강력 매수 | 85점 이상 | 모든 지표가 매우 긍정적 |
| 📈 매수 | 70~84점 | 긍정적 신호 우세 |
| 🟡 관망 | 55~69점 | 방향성 불분명 |
| ⚠️ 주의 | 40~54점 | 부정적 신호 출현 |
| 📉 매도 | 25~39점 | 하락 신호 우세 |
| 🔻 강력 매도 | 25점 미만 | 강한 하락 신호 |

### 분석 지표
1. **RSI (상대강도지수)**
   - 30 이하: 과매도 → 반등 가능
   - 70 이상: 과매수 → 조정 가능

2. **MACD**
   - 골든크로스: 매수 신호
   - 데드크로스: 매도 신호

3. **볼린저 밴드**
   - 하단 터치: 반등 매수 타점
   - 상단 터치: 단기 고점 가능

4. **이동평균선 정배열**
   - 현재가 > 20일 > 50일 > 200일: 강한 상승 추세

### 매수/매도 타점
- **1차 매수**: 볼린저 하단 근처
- **2차 매수**: 지지선 근처
- **손절가**: 지지선 -3%
- **1차 매도**: 볼린저 상단 근처
- **2차 매도**: 저항선 근처

## 🔧 문제 해결

### "모듈을 찾을 수 없습니다" 오류
```bash
pip install -r requirements.txt
```

### 한글이 깨지는 경우
Windows에서 "맑은 고딕" 폰트를 사용합니다. GUI 창에서는 정상 표시됩니다.

### 티커를 복사해도 반응이 없는 경우
- 올바른 형식인지 확인 (예: `AAPL`, `005930.KS`)
- 인터넷 연결 확인
- 프로그램 재시작
