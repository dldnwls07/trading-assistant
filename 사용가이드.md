# 📊 Trading Assistant (QuantCore v3) 사용 가이드

## 🚀 실행 방법 (Web Dashoard)

### 방법 1: 통합 실행 (권장)
프로젝트 폴더의 **`start_web.bat`** 파일을 더블클릭하세요.
- 자동으로 백엔드 API와 프론트엔드 웹 서버가 실행됩니다.
- 실행 후 브라우저에서 `http://localhost:5173`에 접속하세요.

### 방법 2: 수동 실행
1. **Backend**: `python -m uvicorn src.api.server:app --reload`
2. **Frontend**: `cd frontend && npm run dev`

## 🎯 주요 기능 및 사용법

### 1. 지능형 종목 분석
- **검색**: 상단 검색창에 티커(AAPL, TSLA) 또는 한국 종목코드(005930)를 입력하세요.
- **AI 리포트**: 딥러닝 모델이 생성한 전략 보고서와 함께 **패턴 신뢰도(별점)**를 확인하세요.
- **돌파 알림**: 가격이 AI 타겟이나 저항선에 근접하면 화면 상단에 **실시간 알림**이 표시됩니다.

### 2. 전문가용 차트 도구 (TradingView Engine)
- **인터벌 전환**: 1분봉부터 1년봉까지 자유롭게 타임프레임을 변경할 수 있습니다.
- **AI 자동 작도**: AI가 감지한 15종 이상의 패턴(헤드앤숄더, 쐐기형 등)이 차트 위에 자동으로 그려집니다.
- **수동 작도**: 
  - `hline` (가로선): 버튼 클릭 후 차트를 클릭하여 지지/저항선을 직접 그릴 수 있습니다.
  - `Maximize`: 차트를 전체 화면으로 확대하여 정밀 분석이 가능합니다.

### 3. 개인화 테마 (Neural Themes)
- 설정(`Settings`) 버튼을 눌러 원하는 테마를 선택하세요:
  - **Cyber**: 강렬한 사이언 그린 & 블루 (기본)
  - **Stealth**: 차분하고 젠틀한 슬레이트 블루
  - **Bloom**: 화사하고 에너제틱한 핑크 퍼플

## 🌍 지원 시장 및 티커 형식

- **미국(US)**: `AAPL`, `MSFT`, `TSLA` (직접 입력)
- **한국(KR)**: `005930` (삼성전자), `000660` (SK하이닉스) 형태 또는 `005930.KS`
- **일본/유럽 등**: Yahoo Finance 티커 규격을 따릅니다.

## 🛠 문제 해결

### 차트가 멈추거나 에러가 나는 경우
- 브라우저에서 `Ctrl + F5`를 눌러 강력 새로고침을 해주세요.
- 데이터 정렬 오류는 최신 업데이트로 해결되었으나, 네트워크 지연 시 발생할 수 있습니다.

### "npm / uvicorn 명령어를 찾을 수 없음"
- Node.js 및 Python 가상환경 설정을 확인하세요.
- `pip install -r requirements.txt` 및 `npm install` (최초 1회) 설치가 필요합니다.
