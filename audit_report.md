# 🔍 사이트 종합 감사 및 고도화 계획 (초보자 & 전문가 관점)

사용자님의 사이트를 '초보 투자자'와 '기관급 애널리스트' 두 가지 관점에서 정밀 분석한 결과, 다음과 같은 개선 사항을 도출하고 즉시 반영을 시작했습니다.

## 1. 초보자 관점 (Beginner Perspective)
### 🔴 부족한 점:
* **용어 장벽**: RSI, MACD, HHI 등 어려운 용어가 설명 없이 데이터로만 나열됨.
* **해석의 어려움**: 점수가 70점일 때, 이것이 왜 좋은지/나쁜지에 대한 직관적인 가이드 부족.
* **학습 콘텐츠 부재**: 도구를 어떻게 활용해야 할지에 대한 교육적 요소 부재.

### ✅ 개선 완료/진행 사항:
* [x] **AI 트레이딩 사전 구축**: 모든 주요 지표에 대해 초보자용/전문가용 설명을 분리하여 데이터베이스화 (`src/utils/dictionary.py`).
* [x] **동적 도움말(HelpTooltip) 컴포넌트**: 지표 옆의 [i] 버튼을 클릭하면 즉시 쉬운 설명을 볼 수 있는 기능 개발.
* [ ] **대화형 온보딩**: AI 비서가 첫 접속 시 주요 기능을 가이드하는 로직 추가 예정.

## 2. 고수 애널리스트 관점 (Expert Perspective)
### 🔴 부족한 점:
* **리스크 지표 부족**: 단순 수익률 외에 샤프 지수(Sharpe), 베타(Beta) 등 변동성 대비 효율성 분석 부재.
* **데이터 깊이**: 개별 종목 분석은 훌륭하나, 포트폴리오 전체의 시장 민감도 분석이 약함.
* **백테스트 신뢰도**: AI의 예측이 과거에 얼마나 맞았는지에 대한 증거 제시 부족.

### ✅ 개선 완료/진행 사항:
* [x] **전문가급 리스크 엔진**: 포트폴리오 분석기에 **Beta** (시장 민감도) 및 **Sharpe Ratio** (위험 대비 수익 효율) 계산 로직 탑재.
* [x] **상관관계 매트릭스 고도화**: 단순 상관관계를 넘어 시장 지수(^GSPC)와의 공분산 분석 추가.
* [ ] **백테스트 시각화**: 차트 상에 AI의 과거 매수/매도 시점과 이후 결과를 표시하는 기능 연구 중.

## 3. 종합 테스트 결과 요약
* **안정성**: RSI 0.0 오류 해결로 데이터 신뢰도 확보됨.
* **데이터 수집**: KR/US 하이브리드 수집 엔진이 이제 전문가용 지수 데이터를 포함하도록 확장됨.
* **UI/UX**: 프리미엄 디자인은 훌륭하나, 정보의 '친절함'과 '전문성' 사이의 균형을 맞추는 중.
